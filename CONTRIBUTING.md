# Contributing to A2Z Engineering

Thank you for contributing to our corporate website. Please follow the instructions below to submit changes:

## Development Workflow

1. **Fork or Branch**: Create a feature branch off the `main` branch.
   ```bash
   git checkout -b feature/your-feature-name
   ```

2. **Run Locally**: Navigate to the `frontend` directory, install dependencies, and start the development server.
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **Verify Changes**: Before submitting, ensure that the application builds and compile checks pass successfully.
   ```bash
   npm run build
   ```

4. **Pull Requests**: Open a pull request targeting the `main` branch with a description of the bug fixed or feature added.
